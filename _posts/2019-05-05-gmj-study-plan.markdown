# 이 포스트의 목적
미리 한달치의 계획을 짜봐야 어짜피 못하므로,  
다음과 같은 내용들로 기재될 예정이다.

1. 이번달에 풀 예정인 문제 목록 (초기에 기획)
2. 예정엔 없었으나 푼 문제들
3. 중간 중간 '다음에 풀어야징' 하면서 킵 해둘 문제들

푼 문제들은 일정을 업데이트 하면서 문제들에선 삭제됨.
