## 설명

m x n 크기의 알파벳 행렬에서 원하는 단어를 가로, 세로, 대각선에서 찾기.


## 조건

1. 대소문자 구분 안 함
2. 1 <= m,n <=50
3. 찾아야 할 단어 갯수 k가 주어짐 (1 <= k <= 20)
4. 알파벳 이외의 것은 없음.
